# Snake
Snake game using python
 
a main menu gui with start option and exit buttons.
---
<pre>
start => starts the snake game
exit => closes the main menu aka the game
option gui has:
   • snake size
       ► small
       ► medium
       ► large
   • snake color
       ► green
       ► blue
       ► purple
       ► red
       ► orange
       ► yellow
       ► rainbow
   • game size
       ► small
       ► medium
       ► large
   • dark / light mode 
       ► (background = black / white, text = white / black)
</pre>
